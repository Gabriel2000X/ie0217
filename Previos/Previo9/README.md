# Instrucciones para compilado y ejecución 

Las siguientes instrucciones se deben seguir para compilar y ejecutar los archivos del previo 9.   
```
1. Para compilar regex.cpp se debe estar ubicado en la carpeta Previo9 e introducir en la consola lo siguiente:  
g++ -o regex.exe ./regex.cpp    
   
   Se puede cambiar la palabra "regex" de regex.exe si se quiere que el ejecutable a generar   
   tenga otro nombre.    


   Para ejecutar el archvo generado se debe utilizar en la consola la siguiente línea: ./regex.exe   
```
```
2. Para compilar tanto los archivos tanto de la carpeta estudiante como de la carpeta biblioteca se debe ubicar dentro   
de la carpeta donde estén los archivos que se quieren compilar, ya sea estudiante o biblioteca y utilizar la línea para    
linux:   
make    

para windows se debe utilizar:    

mingw32-make   

Posteriormente para ejecutar los archivos generados se debe introducir lo siguiente en la consola:   

Para la carpeta estudiante: ./gestion_estudiantes.exe   

Para la carpeta biblioteca: ./gestion_biblioteca.exe   



```
